# Industrial Robot Arm Simulation - Pick & Place (Unity 3D)

A 3D industrial simulation developed in the **Unity Engine**, featuring a robotic arm performing automated "Pick and Place" operations between two active conveyor belts.

### Academic Background
This project was developed as part of the **"Graphic Lab"** course during my **M.Eng. in Industry 4.0** studies at **Hochschule Albstadt-Sigmaringen**. 

As a Mechanical Engineer transitioning into Industrial Automation, this project serves as a conceptual **Digital Twin**. It demonstrates the intersection of software engineering (C# scripting) and industrial robotics (kinematics, material handling).

## Core Mechanics & Features

* **Robotic Kinematics & Animation:** Programmed the robotic arm to accurately track, pick, and place objects using Unity's animation and transform systems.
* **Conveyor Belt Physics:** Simulated continuous material flow. Objects dynamically move along the belts interacting with physics colliders.
* **Sensor & Trigger Logic:** Implemented virtual sensors (Trigger Colliders) in C# to detect when a product reaches the pick-up zone, prompting the robot to act.
* **State Management:** The robot operates in a continuous loop, mimicking real-world PLC (Programmable Logic Controller) cycle logic within the Unity environment.

## How to Open This Project
To keep the repository lightweight, the `Library` folder has been excluded.
1. **Clone** this repository.
2. Open **Unity Hub** -> Click **Add/Open** -> Select the cloned folder.
3. Allow Unity a few minutes to rebuild the local files.
4. Open the main scene in `Assets/Scenes` and hit **Play**!
